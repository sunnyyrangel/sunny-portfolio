# Portfolio Redesign — Sunny Rangel

## Direction
Clean, minimal, white background. Project images are the only color. Editorial feel.

## Structure
- Homepage: hero, project previews (large/full-width), about blurb, contact footer
- Case study pages: /projects/[slug] — DGW Branded, Disney Pixar, Kaiser Permanente, Sephora, Foster Greatness
- Photography page: /photography

## Colors
- BG: #ffffff / #fafafa
- Text: #171717
- Secondary: #737373
- Borders: #e5e5e5
- Accent: #171717 (black only)

## Typography
- Headings: Instrument Serif
- Body: DM Sans

## Layout
- Nav: name left, links right, no bg, fixed
- Hero: left-aligned, serif name, sans subtitle, one-liner
- Projects: full-width image blocks, title + one-line + category, link to case study
- About: short 2-3 sentences
- Footer/contact: email, LinkedIn, GitHub
